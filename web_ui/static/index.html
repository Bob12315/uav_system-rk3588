机体系单步移动；后端读取当前 LOCAL_NED 和 yaw，换算为 LOCAL_NED 绝对位置目标，并保持当前 yaw。执行前会停止当前 Action 并清理连续控制。

      


      

        
Manual Command

        

          手动命令会直接进入后端命令解析；飞控实发仍受 SEND / 模式 / 解锁状态影响。危险命令请确认后再发送。
        

        

          
          
          
        

        
支持与主命令行相同的命令；复杂命令可在 Config / Logs 页使用完整命令行。

        

          常用命令：
          mode GUIDED
          arm
          land
          mode RTL
          controller all off
